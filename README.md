# DevOps Lab

A reproducible DevOps environment using Vagrant + Ansible + Kubernetes.

## Stack

- Ubuntu 22.04
- Docker
- kind (Kubernetes)
- kubectl
- Helm
- Kyverno-ready environment
- Metrics Server
- Argo CD (GitOps)

---

## Requirements

- Vagrant
- Parallels or VMware
- Ansible

---

## Start environment

```bash
vagrant up
vagrant ssh
```
